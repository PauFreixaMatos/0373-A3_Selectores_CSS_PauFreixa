Selectores CSS

Los selectores CSS se usan para "encontrar" (o seleccionar) los elementos HTML que quieres aplicar estilo.

Podemos dividir los selectores CSS en cinco categorías:

  • Selectores simples (seleccionan por nombre, id o clase)
  • Selectores combinadores (según relación entre elementos)
  • Selectores de pseudo-clase (según un estado concreto)
  • Selectores de pseudo-elementos (estilizan una parte del elemento)
  • Selectores de atributo (por atributo o valor del atributo)

El Selector de elemento en CSS

El selector de elemento selecciona elementos HTML basándose en el nombre del elemento (body, p, h1, h5, a, div, etc.).

El Selector de id en CSS

El selector id usa el atributo id de un elemento HTML para seleccionar un elemento específico.

El id de un elemento es único en la página, así que se usa para seleccionar un único elemento.

Para seleccionar un elemento con un id concreto, escribe el carácter almohadilla (#) seguido del id del elemento.

El Selector de clase en CSS

El selector de clase selecciona elementos HTML que tengan un atributo class específico.

Para seleccionar elementos con una clase concreta, escribe un punto (.) seguido del nombre de la clase.

El Selector Universal en CSS

El selector universal (*) selecciona todos los elementos HTML de la página.
